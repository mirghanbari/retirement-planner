# West Seattle Escape Plan Calculator 🌊🏔️

An interactive financial planning calculator for your 5-year plan to move overseas.

## Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- npm (included with Node.js)

## Setup

```bash
# Install dependencies (only needed once)
npm install

# Start the dev server
npm run dev
```

Then open your browser to **http://localhost:5173**

## Other Commands

```bash
# Build for production (creates a /dist folder you can host anywhere)
npm run build

# Preview the production build locally
npm run preview
```

## What's Inside

Four interactive tabs, all driven by real-time sliders:

- **Overview** — Gauge meters for portfolio, 401k, home equity, and projected monthly income
- **Scenarios** — Conservative / Moderate / Optimal income scenarios side by side
- **Rental** — Unfurnished vs. furnished/corporate lease math for your West Seattle townhome
- **Growth** — Year-by-year portfolio and net worth projections over 10 years
